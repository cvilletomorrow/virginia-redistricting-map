# Virginia Redistricting Map (2021 vs 2026)

This project is a lightweight, static web app that lets users compare Virginia’s 2021 congressional map with a proposed 2026 congressional map. 
The goal is to make it easy for readers to explore how district boundaries shift over time and how those changes affect specific communities.

The 2026 map is provisional and subject to change based on future legal or legislative action.

---
## Features

- Side‑by‑side comparison of congressional districts for 2021 vs 2026.
- Mouseover / tap interaction that highlights districts and shows basic info (district number, geography, etc.).  
- Virginia‑specific “lens” page (`va-lens.html`) that frames the changes for local readers.
